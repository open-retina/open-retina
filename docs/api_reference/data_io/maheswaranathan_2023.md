# Maheswaranathan et al. 2023 Dataset

[Interpreting the retinal neural code for natural scenes: From computations to neurons](https://doi.org/10.1016/j.neuron.2023.06.007), Neuron.

Dataset: [purl.stanford.edu/rk663dm5577](https://purl.stanford.edu/rk663dm5577)
